# Bem-vindo ao Seu App Expo 👋

Este é um projeto [Expo](https://expo.dev) criado usando [`create-expo-app`](https://www.npmjs.com/package/create-expo-app). Siga os passos abaixo para começar e aproveitar ao máximo sua experiência de desenvolvimento.

---

## 🚀 Começando

### 1. Navegue até o Diretório do Projeto

Certifique-se de estar no diretório correto antes de executar os comandos. Use:

```bash
cd quick-table/frontend
```

### 2. Instale as Dependências

Execute o seguinte comando para instalar as dependências necessárias:

```bash
npm install
```

### 3. Inicie o App

Inicie o servidor de desenvolvimento com:

```bash
npx expo start
```

Depois de iniciar, você terá opções para abrir o app em:

- Uma [build de desenvolvimento](https://docs.expo.dev/develop/development-builds/introduction/)
- Um [emulador Android](https://docs.expo.dev/workflow/android-studio-emulator/)
- Um [simulador iOS](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), um ambiente para testar seu app

### 4. Comece a Desenvolver

Comece a editar os arquivos no diretório **app**. Este projeto utiliza [roteamento baseado em arquivos](https://docs.expo.dev/router/introduction) para navegação.

---

## 🔄 Resetar o Projeto

Se você quiser começar do zero, execute:

```bash
npm run reset-project
```

Isso moverá o código inicial para o diretório **app-example** e criará um diretório **app** em branco para seu novo desenvolvimento.
